DSmini help
when first run, DSmini will setup 4 channels to select from. An alert will display to inform you.
use the menu on the upper right to do something. Use set video to expand the video area at the top. choose a channel and select a video. Select the menu option again to collapse it.
Clicking on a profile image from the comments will take you to the replies for that comment.
Clicking on a profile image from the replies will TDS transport you to that comment in the Youtube app.
CLOSE the youtube app before you TDS transport. If it isn't closed first, it wont load a different comment.
Set a status using the select next to the profile image. setting a status back to blank removes the account from your list.
